Connect your domain to Shopify via Cloudflare

A step-by-step guide for setting up your custom domain using Cloudflare's free DNS (with built-in DDoS protection) and linking it to your Shopify store.

1

Sign up for Cloudflare

  1. Go to cloudflare.com and create a free account
  2. Click "Add a site" and type in your domain name (e.g. allseasonplantswa.com.au)
  3. Select the Free plan
  4. Cloudflare will scan for existing DNS records. You can skip or delete anything it finds since the domain is new
  5. Cloudflare will give you two nameservers, something like:
    alma.ns.cloudflare.com
    guy.ns.cloudflare.com

    Write these down (yours will be different to the example above). You'll need them in the next step.

2

Point VentraIP to Cloudflare

  1. Log in to VIPControl
  2. Go to My Services → Domain Management
  3. Find your domain and click the "DNS" button next to it
  4. Click the Custom Nameservers tab, then click Enable
  5. Delete any existing nameservers that are already listed (e.g. VentraIP's default ones)
  6. Enter the two Cloudflare nameservers from Step 1
  7. Save

This takes anywhere from 30 minutes to a few hours to kick in. Cloudflare will send you an email when it's active.

3

Add DNS records in Cloudflare

Once Cloudflare confirms your domain is active:

  1. In Cloudflare, go to DNS → Records
  2. Add these two records:
Type Name Target Proxy status
A @ 23.227.38.65 DNS only (grey cloud)
CNAME www shops.myshopify.com DNS only (grey cloud)
Important: Make sure both records show a grey cloud icon, not orange. Click the orange cloud to toggle it off. Shopify has its own CDN and SSL, so Cloudflare's proxy needs to be off for it to work properly.
4

Connect the domain in Shopify

  1. In your Shopify admin, go to Settings → Domains
  2. Click "Connect existing domain"
  3. Enter your domain (e.g. allseasonplantswa.com.au)
  4. Shopify will verify the DNS records and set up SSL automatically

This can take up to 24 hours but is usually much quicker.

5

Check it works

  • Visit your domain in a browser. You should see your Shopify store.
  • Check that the www version works too (e.g. www.allseasonplantswa.com.au)
  • The padlock icon (HTTPS) should appear automatically once Shopify provisions the SSL certificate.

If something goes wrong

"Domain not verified" in Shopify: Double-check the DNS records are exactly right, and that the proxy is off (grey cloud, not orange). Give it a bit more time.
SSL errors / no padlock: These usually resolve within an hour once Shopify picks up the DNS. Don't enable "Always Use HTTPS" in Cloudflare; Shopify handles that itself.
