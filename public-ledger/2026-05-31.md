### Correction caught in review

The first cut of the species-page box posted `action:'watch'` with a species
slug, which would have revived **species-level watches** (deliberately removed
in commit `3f89a09`; the pipeline notes them deprecated 2026-04-19 as too strict
to ever fire). The server has no `watch` handler and `send_species_alerts.py` is
not run by the pipeline, so that path was dead. Reworked the box into a
**daily-digest subscribe** form (`action:'subscribe'` + state), which is the
surviving, working capture channel and itself flags back-in-stock items, price
drops, and new arrivals. Per-variety watches (the kept low-noise feature, via
the bell links to `/variety/<slug>.html`) are unchanged.
