# Public Ledger — 2026-06-01

## When to Plant calendar: re-homed as a maintainable, evidence-backed builder

/when-to-plant.html was built once in March 2026 (DEC-100) and then left as an orphaned
static page: no builder regenerated it (its file mtime had been frozen since 28 March), its
50-species planting calendar was rendered only in client-side JavaScript (so search engines
saw an empty table), and it had no citations, an unversioned stylesheet, only four climate
zones, and em/en dashes.

### Shipped (committed, pushed, merged, deployed, verified live)

New `tools/scrapers/build_when_to_plant.py`, modelled on the companion-guide builder and
wired into `run-all-scrapers.sh` so the page now regenerates every day:

1. **Server-side rendered calendar** (50 species). The table is now in the HTML, so it is
   crawlable. This was the biggest weakness of the old page.
2. **Shared layout** (treestock_layout): sticky nav, footer, and a versioned styles.css
   (the old page referenced an unversioned /styles.css).
3. **Five Australian growing zones**, adding the arid / dry-inland zone, with a
   progressive-enhancement zone filter that works without JavaScript.
4. **Per-section citations plus a References block** from authoritative Australian sources
   (ABC Gardening Australia, Sustainable Gardening Australia, NT Government, the Bureau of
   Meteorology, DPIRD WA, Bayside Council, Daleys), every URL verified to resolve.
5. **FAQPage JSON-LD**, Open Graph tags, and /species/ links validated against
   fruit_species.json so they cannot 404. No em or en dashes.

### How the content was checked

The planting facts were researched and adversarially verified against Australian
authorities via a fan-out workflow (17 agents, roughly 875k tokens, 614 web fetches). The
existing 50-species table was then audited against the verified facts: no tree-killing
errors were found, and four fixes were applied (a frost warning for dragon fruit in
temperate zones, apple tagged subtropical to match its low-chill note, arid tags for loquat
and apricot, and a clearer bare-root note for blueberries).

### Verification
